How to install Git and Connect to GitHub Account

First of all, we need to understand what Git and GitHub are, and their purposes.

Git - Global Information Tracker

Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently.

Git is used to track changes in the source code, enabling multiple developers to work together on non-linear development.

GitHub - Git + Hub

GitHub is a platform for hosting and managing code using Git, a version control system. It helps developers collaborate, track changes, and store code in repositories.

It also serves as a hub for open-source projects, allowing developers worldwide to contribute, share ideas, and learn from each other's work.

How do Git and GitHub work together?

When we upload files to GitHub, we'll store them in a "Git repository." This means that when we make changes (or "commits") to files in GitHub, Git will automatically start to track and manage your changes.

Relationship between Git and GitHub

GitHub, meanwhile, serves as a host for Git repository teams to store their code in a centralized location. While Git is a tool that's used to manage multiple versions of source code edits that are then transferred to files in a Git repository, GitHub serves as a location for uploading copies of a Git repository.

Procedure to install Git and connect to GitHub account

1. Install Git:

Download and install Git from the official Git website.

During installation, select the default options unless you have specific preferences.

2. Set Up Git:

Open your terminal or command prompt and configure Git with your GitHub account:

git config --global user.name "YourGitHubUsername"

git config --global user.email "YourEmail@example.com"

3. Create a Repository on GitHub:

Log in to your GitHub account.

Click on the + icon in the top-right corner and select New repository.

Give it a name, optionally add a description, and click Create repository.

4. Initialize Git in Your Project Folder:

Navigate to your project folder using the terminal:

cd path/to/your/project

Initialize Git:

git init

5. Connect to Your GitHub Repository:

Add your GitHub repository as a remote:

git remote add origin https://github.com/YourGitHubUsername/YourRepositoryName.git

6. Add and Commit Your Files

Stage your files:

git add .

Commit your changes:

git commit -m "Initial commit"

7. Push Your Code to GitHub:

Push your changes to GitHub:

git branch -M main

git push -u origin main
